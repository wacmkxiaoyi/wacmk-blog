# Comments, Feedback, Stars, and Rewards

## Commenting

Comments are available only when the site allows them.

The site can configure commenting as:

- enabled for normal eligible users
- disabled globally
- limited to VIP users

## Comment Structure

The platform supports:

- top-level comments
- replies under a top-level comment

It does not behave like an unlimited-depth nested forum.

Replies stay inside a controlled two-level thread model so discussions remain readable.

## Comment Editor Features

The comment editor can support:

- Markdown
- image upload
- video upload when allowed
- tables
- callouts
- side-by-side preview
- emoji support

## Editing and Deleting Comments

Users can manage their own comments from `My comments`.

Staff can moderate comments from the management workspace.

## Rate Limiting

Comment posting can be rate-limited. If a user posts too quickly, the site may require a wait before the next comment.

## Comment Feedback

Comments support positive and negative feedback.

This behaves like a toggle:

- click positive to set positive feedback
- click negative to set negative feedback
- click the same choice again to remove it

## Article Feedback

Article pages can support:

- like
- dislike
- star or unstar

## Book Stars

Books can also be starred in list views.

## First Comment Reward

Depending on site settings, a user can receive a first-comment reward on an article.

Important details:

- it is once per user per article
- replies can count if they are the user's first comment on that article
- reward can include base values and VIP bonus values

## Daily Login Reward vs Comment Reward

These are separate systems.

- daily login reward is based on the first successful login of the day
- first comment reward is based on the first comment on a specific article
