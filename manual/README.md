## Important Reading Notes

Many features are conditional. What a user can see or do depends on:

- whether the user is logged in
- whether the user is a normal user, VIP user, staff user, or superuser
- whether the site administrator enabled that feature
- the access rules of a specific article, book, or attachment

## Key Concepts Used Throughout the Manual

- `public`: immediately accessible
- `private`: restricted to the owner or privileged staff
- `conditional`: accessible only after satisfying rules
- `unified`: VIP users and non-VIP users use the same rule set
- `standalone`: VIP users may use a separate VIP rule set

The user manual explains these concepts in more detail in the reference section.
