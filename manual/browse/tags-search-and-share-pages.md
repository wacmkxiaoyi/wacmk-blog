# Tags, Search, and Share Pages

## Tags

The tag directory shows tags that currently have visible content.

Each tag page shows:

- the tag name
- post count
- the visible articles connected to that tag

## Tag Visibility Note

Tags are not simply a master list of every tag in the database. Hidden-only or inaccessible tags may not appear.

## Dedicated Search Results

The `/search/` page is a dedicated article-result page.

It can search article-related fields such as:

- article title
- article summary
- article body
- related book name
- tag name
- author name

## Empty Search Behavior

If no search keyword is entered, the dedicated search page does not behave like a full browse-all list.

## Share Pages

Share pages allow selected public content to be opened without normal login-based reading flow.

There are two main share-page types:

- article share page
- book share page

## Important Share-Page Rules

- share links only exist for eligible public content
- share links are not a bypass for protected conditional content
- interaction is more limited than on normal authenticated pages

## Common Share-Page Differences

Compared with normal signed-in pages, share pages may:

- hide or disable reaction features
- keep comments visible but restrict interaction
- disable clickable tag-navigation behavior
- use a simpler page toolbar
